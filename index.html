About

Hello, this webpage is the gateway of my progression in developing webpages. In my class of Intro to Internet Computing, I learned HTML5, CSS, Bootstrap, JavaScript and jQuery. In order to help grasp our understanding of the concept we were assigned a few projects to work on to help on a particular subject in web development. Below you can read a summary about each project and view them on a seperate tab by clicking on the images.

Project 1

Here you will see my first encounter with web development. In this project we deal with the basics of HTML5 like adding tables, ordered lists, headings, etc. Using these tools I created a simple website that displays a small portfolio of me.

Project 2

In this project we learned CSS and how we can implement Bootstrap on to our website to make a visually pleasant layout for the user. I used this project as a record to see the difference from the first project where both contain the same contents of my portfolio.

Project 3

In this project we learn about Javascript and how we can make an interacting webpage. In doing so, I created a simple calculator were I use JavaScript to take a user's input and perform some calculations and display to the user.

Project 4

In this last project, I learned about jQuery and how it's functionality allows web developers to create a dynamic webpage with less code (than of JavaScript). In doing so, I edited a simple tic-tac-toe template in order to provide more visuals to the user interface.
